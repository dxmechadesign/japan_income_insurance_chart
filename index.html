📈 1970年〜2000年：成長期

社会保険料負担率: 3.5% → 10.5%（3倍増）

年収中央値: 150万円 → 400万円（2.7倍増）

出生数: 193万人 → 119万人（38%減）

経済成長により年収は大幅上昇したが、同時に社会保険制度の充実で負担も増加。出生数は既にこの時期から大幅減少。

⚠️ 2000年〜2024年：転換期

社会保険料負担率: 10.5% → 14.9%（4.4%増）

年収中央値: 400万円 → 407万円（ほぼ横ばい）

出生数: 119万人 → 68.5万人（42%減）

デフレ長期化で年収停滞の中、高齢化進行により保険料負担のみ急増。出生数は加速度的に減少。

🚨 2024年〜2050年：危機期

社会保険料負担率: 14.9% → 22.3%（予測7.4%増）

年収中央値: 407万円 → 340万円（予測67万円減）

出生数: 68.5万人 → 51万人（予測25%減）

年収減少と保険料急増のダブルパンチ。出生数は50万人台まで落ち込み、社会保障制度は根本的見直しが不可避。

💸 80年間の変化総括

保険料負担率: 6.4倍増（3.5% → 22.3%）

年収中央値: 2.3倍増（150万円 → 340万円）

実質負担額: 約15倍増（5万円 → 76万円）

出生数: 73%減（193万人 → 51万人）

80年間で日本社会は完全に様変わり。負担は激増、人口は激減の厳しい現実。
